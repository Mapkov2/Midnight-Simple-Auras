Midnight Simple Auras.
/msa lock unlock etc 

Fully standalone lightweight personal cooldown tracker designed for the Midnight (12.0) API.
Tracks your spell cooldowns and trinkets with a simple WeakAura-like display — secure, fast, and easy to configure.

The addon will be working alongside MSUF. 

Addon will namechange in the next release of MSUF

Features
Track any personal spell cooldown (add by SpellID)
Track any personal buff that is based on a cooldown trigger 
Trinket Slot 13 / 14 support with full positioning and resizing
Secure cooldown passthrough using Blizzard’s native cooldown API
Advanced anchroing
Drag-to-move icons, adjust size, or nudge by ±1 pixel
Masque integration for custom icon skins
SharedMedia font support with live preview


Standalone version will remain available for minimal setups.

Optional Dependencies
Masque (icon skinning)
LibSharedMedia-3.0 (extra fonts)
